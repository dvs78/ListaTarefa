# ListaTarefa ✅

Aplicação web para cadastro e controle de tarefas pessoais.  
Criado com React no front-end e Node.js no back-end.

## Tecnologias

- React + Vite
- Node.js + Express
- PostgreSQL
- Toastify, FontAwesome, etc.

## Como rodar

```bash
cd front-end
npm install
npm run dev
```

```bash
cd back-end
npm install
node index.js
```
